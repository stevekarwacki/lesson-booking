# Admin Overview

This guide is a quick map of everything available to you as an admin. Use it to orient yourself before diving into the detailed how-to guides for each area.

## Your Navigation Menu

As an admin, your navigation bar gives you access to six main areas:

| Nav item | What it's for |
|---|---|
| **Users** | Create and manage all user accounts — students, instructors, and other admins |
| **Calendar** | Today's lessons at a glance; 7-day schedule grid; book or reschedule lessons on behalf of students |
| **Bookings** | Full paginated lesson history across all instructors; search and filter by instructor, student, or date |
| **Packages** | Create and manage the lesson credit packages and memberships students can purchase |
| **Settings** | Configure everything about how the application runs — business info, email, branding, and more |
| **Account** | Your own profile and account preferences |

---

## Users

**Navigation: Users**

This is where you manage the people in your system. You can:

- Create new student, instructor, and admin accounts
- Search and filter users by role or name
- Open any user's detail modal to manage their profile, availability, bookings, subscriptions, and account status

The user detail modal has several tabs that appear based on the user's role:

- **Profile** — edit a student's contact info, address, and minor/parent details
- **Instructor Details** — create or edit the instructor's public profile, specialties, and rate
- **Availability** — manage an instructor's weekly schedule and blocked time
- **Bookings** — view and manage a student's lesson history; reschedule, cancel, or refund
- **Memberships** — view and manage a student's subscription; cancel, reactivate, or grant a complimentary plan
- **Account** — approve/unapprove access, change roles, set payment overrides, or delete the user

> **New student accounts need approval before the student can book lessons.** Until you approve them, they can log in and complete their profile, but cannot access booking or payment features.

---

## Calendar

**Navigation: Calendar**

The Calendar page is your operational view of the day. At the top, a **Today's Bookings** panel shows all lessons scheduled for today. Below that, the weekly schedule grid lets you:

- Browse any instructor's schedule week by week (up to 8 weeks back, unlimited forward)
- Click an open slot to book a lesson for a student
- Click a booked slot to reschedule it

The grid also shows blocked time from Google Calendar (if configured) so you can see the instructor's full availability picture.

---

## Bookings

**Navigation: Bookings**

The Bookings page shows the complete lesson history with filtering and pagination. As an admin you can filter by instructor, student, and date range — useful for looking up a historical booking or reviewing a student's full lesson record. Reschedule, cancel, and refund actions are available directly from the list.

---

## Packages

**Navigation: Packages**

Packages define what students can purchase — both one-time credit bundles and recurring memberships. You can:

- Create new packages with a name, price, number of credits, and lesson duration
- Edit existing packages
- Delete packages that are no longer offered

> Changes to packages only affect future purchases. Students who have already bought a package keep what they paid for.

---

## Settings

**Navigation: Settings**

Settings is tabbed into six sections:

| Tab | What you configure |
|---|---|
| **Business Information** | Company name, contact details, address, social links, and weekly business hours |
| **Theme & Branding** | Color palette, custom colors, and your logo |
| **Lesson Settings** | Default lesson duration and payment policies (in-person payments, card payments on order-on-behalf-of) |
| **Email Settings** | How the system sends email (SMTP, Gmail OAuth, or disabled); email templates |
| **Calendar Settings** | Google Calendar integration method (Service Account, OAuth, or disabled) |
| **Advanced Settings** | File storage configuration (local vs. cloud) |

## Quick reference: common tasks

| Task | Where to go |
|---|---|
| Add a new user | Users → Add User button |
| Approve a new student | Users → find student → Account tab → toggle Approved |
| Edit a student's profile | Users → find student → Profile tab |
| Set up an instructor | Users → find instructor → Instructor Details tab; then Availability tab |
| Book a lesson for a student | Calendar → click an open green slot |
| See today's lessons at a glance | Calendar → Today's Bookings panel |
| Browse the full lesson history | Bookings |
| Search for a specific booking | Bookings → filter by instructor, student, or date |
| Reschedule a lesson | Bookings → find lesson → Reschedule, or Calendar → click the blue slot |
| Cancel a lesson | Bookings → find lesson → Reschedule → Cancel Booking |
| Refund a lesson | Bookings → find lesson → Refund |
| Manage packages/pricing | Packages |
| Change your logo or colors | Settings → Theme & Branding |
| Configure email delivery | Settings → Email Settings |
| Set up Google Calendar sync | Settings → Calendar Settings |
| Change business hours | Settings → Business Information |
