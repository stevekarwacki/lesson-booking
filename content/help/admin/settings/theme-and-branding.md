# Customize Your Theme and Colors

Change the primary color used throughout the app — buttons, links, and accents — to match your brand.

## Where to find it

> Settings → Theme & Branding

## Using a preset palette

1. Click any of the five color swatches to select a palette.
2. The preview updates instantly.
3. Click **Save Changes**.

The available palettes are: Forest Green (default), Ocean Blue, Sunset Orange, Royal Purple, and Charcoal.

## Using a custom color

1. Click the color picker or type a hex code (e.g. `#1a73e8`) in the text field.
2. If the color doesn't meet accessibility contrast standards, a warning appears — you can still save it, but it may be hard to read.
3. Click **Save Changes**.

## Resetting to defaults

Click **Reset to Defaults** to return to Forest Green. This takes effect immediately.
