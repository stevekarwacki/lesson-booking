# Managing Users

The Users page is where you manage every account in the system — students, instructors, and other admins.

## The user list

The list shows all users. Use the search bar and role filter to find specific people quickly.

→ [Search and filter users](search-and-filter.md)

## Adding users

Use the **Add User** button to create any account type directly, without the person needing to sign up themselves.

→ [Create a new user account](create-users.md)

## The user detail modal

Click any user's name to open their detail modal. The tabs available depend on the user's role:

| Tab | Roles | What you can do |
|---|---|---|
| **Profile** | Student | Edit contact info, address, and minor/parent details |
| **Instructor Details** | Instructor | Create or edit the instructor's public profile and rate |
| **Availability** | Instructor | Set weekly schedule and blocked time |
| **Bookings** | Student | View lesson history; reschedule, cancel, or refund |
| **Memberships** | Student | View and manage subscription; cancel, reactivate, or grant complimentary plan |
| **Account** | All | Approve access, change role, set payment overrides, delete account |
