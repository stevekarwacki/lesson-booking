# Create a New User Account

Add students, instructors, or other admins directly from the Users page.

## Where to find it

> Users → Add User button

## Steps

1. Click **Add User**.
2. Enter the new user's name, email address, and a temporary password.
3. Select a role: **Student**, **Instructor**, or **Admin**.
4. Click **Create User**.

## Notes

- New student accounts are not approved by default. You'll need to [approve them](approve-students.md) before they can book lessons.
- New instructor accounts can be used immediately, but won't appear on the booking calendar until you [set up their profile and availability](instructor-profiles.md).
- The user will need to change their password on first login if you use a temporary one.
