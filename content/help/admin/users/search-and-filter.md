# Search and Filter Users

Quickly find any user by name, email, or role.

## Where to find it

> Users

## Filter tabs

The tab bar at the top of the user list filters by role:

| Tab | Shows |
|---|---|
| All | Everyone |
| Students | Students only |
| Instructors | Instructors only |
| Admins | Admin accounts only |
| Unverified | Students who haven't completed their profile or haven't been approved yet |

## Search

Type a name or email in the search bar. Results filter in real-time and work within whichever tab is active.

## Open a user's profile

Click **Manage** on any user to open their detail modal.

## Notes

- The **Unverified** tab is the quickest way to find students waiting for approval.
- Your search query stays visible while you open and close modals — clear it manually or switch tabs to reset.
